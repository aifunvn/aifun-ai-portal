🔧4. Danh sách lỗi đã phát hiện và xử lý
#LỗiNguyên nhânGiải phápFile
BUG-01 GAS URL mismatch config.js trỏ sai URL deployment cũ, nested quotes lỗi syntax Rewrite config.js với URL đúng, single quotes config.js
BUG-02 Script 404 — không tải được index.html load scripts từ subdirectory aifun-skill-engine/ không tồn tại Sửa toàn bộ 4 path về root (CSS + 3 JS) index.html
BUG-03 getSkillForm is not defined skill-forms.js bị truncate tại dòng 39, thiếu function + window exports Viết lại hoàn chỉnh skill-forms.js với 6 skills + exports skill-forms.js
BUG-04 Nút submit không click được (Cốc Cốc) type=submit + form onsubmit không fire trong Cốc Cốc Đổi sang type=button + onclick="SkillEngine._onFormSubmit()" skill-engine.js
BUG-05 Cannot read properties of undefined (reading 'map') form tồn tại nhưng aiOutputSections undefined → undefined.map() Thêm guard: (form && form.aiOutputSections) ? ... : fallback skill-engine.js
BUG-06 CORS "Failed to fetch" Content-Type: application/json trigger OPTIONS preflight; GAS không handle preflight Đổi sang Content-Type: text/plain (simple request, không cần preflight) skill-engine.js
BUG-07 Claude API 404 Model claude-3-5-sonnet-20241022 đã deprecated Cập nhật AIProvider.gs: CLAUDE_MODEL = 'claude-sonnet-4-6' AIProvider.gs (GAS)
BUG-08 Logger không ghi Sheets (silent fail) SpreadsheetApp.openById() throw "Illegal spreadsheet id" bên trong try-catch → lỗi bị nuốt Dùng DriveApp.getFilesByName('AIFUN DATA CENTER') thay openById Logger.gs (GAS)
BUG-09 CEO Dashboard hiển thị 0 GAS getStats đọc sai index cột do column layout thực tế khác với thiết kế ban đầu Tính stats client-side từ getDocuments, _computeStats() với date parsing index.html
BUG-10 Word Export lỗi với nội dung dài Content được encode inline trong onclick attribute → bị cắt hoặc break do ký tự đặc biệt Lưu content vào window._seLastContent, _downloadDocx() đọc từ global skill-engine.js
